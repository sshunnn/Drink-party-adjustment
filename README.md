# 華麗なる居酒屋 - 飲み会日程調整アプリ

シンプルかつ現代的なデザインを採用した、飲み会の日程と場所を調整するためのWebアプリケーションです。

## 特徴
- **グラスモーフィズムデザイン**: すりガラスのような半透明の要素を取り入れたモダンなUI。
- **代表者/参加者モードの分離**: URLクエリ (`?admin=true`) により、代表者と参加者の画面をシンプルに分離。
- **カレンダー・プルダウン選択**: 直感的な日程選択や、参加者の二重投票を防ぐ機能。
- **厳格な投票チェック**: すべての候補に対する「〇」「×」の投票を必須化し、漏れを防止。
- **バックエンド不要**: データはブラウザの LocalStorage に保存されるため、サーバーなしで動作するデモアプリとして機能します。

## 起動方法

1. 依存パッケージのインストール
```bash
npm install
```

2. 開発用サーバーの起動
```bash
npm run dev
```

ブラウザで `http://localhost:5173/` にアクセスしてください。

- **代表者モード（作成・管理）**: `http://localhost:5173/?admin=true`
- **参加者モード（投票のみ）**: `http://localhost:5173/`

## 技術スタック

- React (TypeScript)
